Walkability vs Proximity — Urban Mobility Research

International research collaboration between Tecnológico de Monterrey (Mexico) and Universidad EAFIT (Colombia), funded by Volvo Research and Educational Foundations (VREF).

Developed both frontend design and backend data exploration modules for an interactive Streamlit application measuring urban walkability in Medellín and Guadalajara.

The project introduces a novel Walkability Index integrating:

  • Population density
  • Intersection density
  • Land-use mix
  • Vegetation index
  • Slopes
  • Sidewalk presence
  • Citizen perception surveys

Combines objective urban indicators with subjective pedestrian perception to better understand mobility decisions in Latin American cities.

CorAI-AI — AFDB Supervised Rhythm Classification

Tech: Python, Scikit-learn, PhysioNet, RandomForest

End-to-end machine learning pipeline for supervised cardiac arrhythmia classification using the MIT-BIH Atrial Fibrillation Database (AFDB).

Classified four rhythm types: AFIB, AFL, AV Junctional (J), and Normal rhythms.

Key components:

  • Advanced ECG feature extraction (temporal, HRV, QRS morphology, wavelets)
  • R-peak detection (Pan-Tompkins inspired)
  • RandomForest classifier with balanced weights
  • 75/25 stratified split
  • Feature importance analysis & PCA visualization

Designed for reproducible research using both streaming PhysioNet access and local file modes.
